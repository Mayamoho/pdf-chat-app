This is my first rag app which is a pdf-chat-app which mainly answers different questions about your uploaded pdf. 
To run the app:
On linux terminal/wsl terminal-
Clone the repository.
Go to the project folder.
Build a virtual environment using python3 -m venv venv
Go to the environment using source venv/bin/activate
Install the required packages from requirements.txt using pip install -r requirements.txt
Run the app using python3 app.py.
